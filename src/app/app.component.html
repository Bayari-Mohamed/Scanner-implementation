About

ngx-barcode-scanner is an Angular wrapper for quaggajs. It provides 2 components to read barcode with a camera in a livestream :
  • barcode-scanner-livestream: a simple livestream component, that can be integrated in any page or component
  • barcode-scanner-livestream-overlay: an overlay livestream component
